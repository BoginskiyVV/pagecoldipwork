4.1. Парсер sqlparse.
Работа с данным парсером начинается с установки (открыть терминал):
"$ pip install sqlparse"



Последнюю выпущенную версию можно получить из индекса пакетов Python (PyPI).
Для извлечения и установки модуля запустите общесистемный распаковщик.



Модуль sqlparse предоставляет три простые функции на уровне модуля для решения некоторых распространенных задач при работе с SQL-инструкциями. В этом разделе показано несколько простых примеров использования этих функций.
Давайте начнем с разделения строки, содержащей один или несколько операторов SQL, на список отдельных операторов с помощью функции split():



Конец инструкции определяется по появлению точки с запятой. Точки с запятой в определенных конструкциях SQL, таких как BEGIN ...
ТОРЦЕВЫЕ блоки корректно обрабатываются механизмом разделения.
Инструкции SQL можно украсить с помощью функции format().



В этом случае все ключевые слова в данном SQL написаны в верхнем регистре, а отступ изменен, чтобы сделать его более читабельным. Ознакомьтесь с форматированием инструкций SQL для получения полной информации о поддерживаемых параметрах, заданных в качестве аргументов ключевых слов для этой функции. Прежде чем перейти к более подробному рассмотрению внутреннего представления SQL-инструкций, вы должны знать, что этот синтаксический анализатор SQL намеренно не выполняет проверку подлинности. Он предполагает, что данные входные данные представляют собой, по крайней мере, какой-то вид SQL, а затем пытается проанализировать как можно больше, не делая слишком больших предположений о конкретном диалекте или фактическом утверждении.
